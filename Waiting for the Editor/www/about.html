This app displays data about wait times collected by the APA Journal Survey.

  1. The scatterplots expose any changes over time in a journal's turnaround times.
  2. Ridgeplots show estimates of probable wait times.
The ridgeplots show more directly what we're intersted in, but they can also be misleading, if a journal has recently improved its turnaround times for example. Some journals have gotten significantly better/worse in recent years. Read this post for more.

Other important caveats:

  • This is volunteer data, hence biased by self-selection. One such bias is that accepted submissions are overrepresented. You can filter out accepted submissions to help with this issue.
  • The cap on review times squashes the data, rather than filter it. When the cap is set to 18 months, all longer review times are modified to 18 months.
  • Some data is filtered out because it appears to be erroneous. See the discussion under "Timeline" here.
